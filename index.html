What is JavaScript?

Simply put JavaScript is the language that makes your web page interactive. HTML marks up your web page with content, CSS designs your page, and JavaScript manipulates and adds control.



Why JavaScript?

JavaScript is a very powerful language that is capable of many things. Here are a list of just some of the great things you can accomplish with Javascipt:

  • Add animation to your website.
  • Create interactive content, like games.
  • Update web page content off the bat.
